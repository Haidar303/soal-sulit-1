Fungsi:
Menghitung nilai rata-rata dari sejumlah nilai ujian siswa.

Cara kerja:

Pengguna memasukkan jumlah siswa.

Program meminta nilai tiap siswa satu per satu.

Semua nilai dijumlahkan lalu dibagi jumlah siswa.

Rata-rata ditampilkan.
